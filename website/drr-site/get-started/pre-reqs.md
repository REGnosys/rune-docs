# Pre-reqs
What you'll need to get started with DRR:

- **CDM data:** Regulatory data converted to the Common Domain Model (CDM).
- **Rune DSL:** An understanding of the domain-specific language used for modelling DRR logic.
- **Execution engine:** e.g. Rosetta platform or a propreitary engine capable of processing DRR.
