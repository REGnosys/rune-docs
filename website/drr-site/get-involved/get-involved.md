# Get involved
There are several ways to get involved with the DRR community. 
<br>
## Contributing to DRR
You can contribute directly to DRR – [view our guidelines](LINK).
<br>
## Contributing to DRR documentation
Found an issue in the documentation? Let us know so we can improve it for everyone – [view our guidelines](LINK).
<br>
## GitHub
DRR primarily resides in the [DRR GitHub repository](LINK). 
<br>
## Mailing list
Our mailing list can help keep you up to date on changes and improvements to DRR. [Join the mailing list](LINK).
<br>
## Meetings
You can also sign up to attend DRR working groups and other meetings – [sign up](LINK).
<br>
## Need help?
If you need a little help getting started or have technical difficulties accessing repositories, mailing lists or meetings, [let us know](LINK).
